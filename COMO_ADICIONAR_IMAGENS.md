# Como Adicionar Imagens aos Produtos

## 📸 USANDO POSTIMAGES (Recomendado)

### Passo a passo completo:

1. **Acesse**: https://postimages.org

2. **Clique em "Choose images"** (botão azul no centro)

3. **Selecione a foto** do produto do seu computador

4. **Aguarde o upload** (alguns segundos)

5. **IMPORTANTE**: Na página de resultado, você verá várias opções de links

6. **Copie o link em "Direct link"** (é o terceiro campo)
   - O link será algo como: `https://i.postimg.cc/abc123/foto.jpg`

7. **Cole no campo "Imagem (URL)"** no painel admin

8. **O preview aparecerá automaticamente** se o link estiver correto

9. **Clique em "💾 Salvar"**

---

## ✅ Link Correto do PostImages:

```
https://i.postimg.cc/abc123/foto.jpg
```

✅ Começa com: `https://i.postimg.cc/`
✅ Termina com: `.jpg`, `.png` ou `.webp`

---

## ⚠️ Problemas Comuns:

**Preview não aparece?**
- Verifique se copiou o "Direct link" (não o "Share link")
- O link deve começar com `https://i.postimg.cc/`
- Teste: cole o link no navegador, deve abrir SÓ a imagem

**Imagem sumiu depois de um tempo?**
- PostImages gratuito mantém imagens por tempo indeterminado
- Se sumir, faça upload novamente

---

## 💡 Dicas:

- Tire fotos com boa iluminação
- Use fundo neutro (branco ou claro)
- Centralize o produto na foto
- Tamanho ideal: 800x800px ou maior
- Formato: JPG (mais leve) ou PNG (melhor qualidade)

---

## 🔄 Para trocar uma imagem:

1. No painel admin, clique em "✏️ Editar" no produto
2. Faça upload da nova foto no PostImages
3. Copie o novo "Direct link"
4. Cole no campo "Imagem (URL)"
5. Salve
